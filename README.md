# Todo List App
## Description
Very basic todo list app that allows for creating todo's, editing them, checking off items, and deleting them. At this time it does not support account creation.
## Technologies
### Backend
* Node.js
  * Express
* Mongo Db
  * Mongoose ODM
### Frontend
* HTML
* CSS
* Javascript
## Development Plan
* Migrate to SQL
* Add user creation feature
